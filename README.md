#  Réseau Étudiant

Bienvenue sur le dépôt du projet **Réseau Étudiant**. Cette application est conçue pour faciliter les interactions et la gestion au sein de la communauté étudiante.

##  Fonctionnalités
- **Gestion des membres** : Inscription et profils étudiants.
- **Réseau social interne** : Publication de contenus et échanges.
- **Architecture robuste** : Utilisation de Middlewares pour la sécurité et la gestion des requêtes.

##  Technologies utilisées
- **Backend** : Node.js, Express
- **Base de données** : PostgreSQL 
- **Gestion de version** : Git & GitHub

## Installation
Pour lancer le projet localement :

1. Cloner le dépôt :
   `git clone https://github.com/sagesse12/R-seau-Etudiant.git`
2. Installer les dépendances :
   `npm install`
3. Lancer le serveur :
   `npm start`
